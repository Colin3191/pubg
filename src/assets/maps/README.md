# 地图图片说明

请在此目录下放置 PUBG 地图图片文件。

## 需要的图片文件：

1. `erangel.jpg` - 海岛地图
2. `miramar.jpg` - 沙漠地图
3. `sanhok.jpg` - 雨林地图
4. `vikendi.jpg` - 雪地地图
5. `karakin.jpg` - 卡拉金地图
6. `taego.jpg` - 褐地地图

## 图片要求：

- 格式：JPG 或 PNG
- 尺寸：建议至少 1920x1080 像素，越高清越好
- 比例：建议保持原始地图比例

## 如何获取地图图片：

1. 从 PUBG 官方网站下载
2. 从游戏截图
3. 从互联网搜索高清地图图片

## 临时方案：

如果没有准备好图片，可以：
1. 使用占位图片服务（如 placehold.co）
2. 在 `src/data/maps.ts` 中修改图片路径为在线图片URL
